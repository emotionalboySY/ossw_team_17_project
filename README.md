# 1. Download .apk file

[snakeGame.zip](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/d0fce317-3947-468a-b98a-3815e67f5468/snakeGame.zip)

- click **EXIT** to return to the main screen.
